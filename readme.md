# Monitoring Stack

## Deploy:
1. Deploy da stack no Portainer
2. Execute: `./setup.sh`

## URLs:
- Prometheus: http://localhost:19090
- Grafana: http://localhost:13000 (admin/admin123)
- Node Exporter: http://localhost:19100

## Estrutura:
- `docker-compose.yml` - Stack principal
- `setup.sh` - Script de configuração
- `prometheus.yml` - Referência de configuração
